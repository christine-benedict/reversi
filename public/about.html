About this site

This is a fun little site made by  Christine Benedict  for the Masters in Human-Computer Interaction and Design course in Interactive Design at UC Irvine.

Single player mode was developed in collaboration with  Serena Epstein.

The underlying background for the site is attributed to Marvel Comics and was a spread from one of the X-Men series featuring Storm. All others are from non-attribution sources.

Game Tech

  • The site was designed using HTML for the base structure.
  • CSS Flexbox was used for the layout styling.
  • Animations were done with a combination of CSS and SVG SMIL.
  • Functionality was accomplished using Javascript and the JQUERY library.
  • Multiplayer functionality was accomplishd using Socket.io.
